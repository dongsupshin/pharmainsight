PromoComply Assets Directory

This directory contains application assets including:

- AppIcon.ico: Main application icon (32x32 or larger)
- Square150x150Logo.png: Store tile (150x150px)
- Square44x44Logo.png: App bar icon (44x44px)
- SplashScreen.png: Splash screen (620x300px minimum)
- StoreLogo.png: Store logo (50x50px)

To create actual icons:
1. Design the icons in your preferred graphics tool (e.g., Figma, Adobe XD, Inkscape)
2. Export in the required formats and dimensions
3. Place in this directory
4. Reference in Package.appxmanifest and .csproj files

Current setup uses placeholder paths. The application will function without these images but will show warnings in the MSIX packaging process.

Recommended icon characteristics:
- Use the PromoComply brand colors: Primary #1B4965, Accent #5FA8D3
- Incorporate elements representing compliance, pharmaceutical industry
- Ensure high contrast and clarity at small sizes
- Use professional, clean design appropriate for enterprise/healthcare sector
